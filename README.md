# Imagens TikZ — Colégio Eleve

Repositório público exclusivo para os PNGs finais aprovados produzidos pelo pipeline TikZ do Colégio Eleve.

## Estrutura

```text
<disciplina>/<ano-serie>/<titulo-do-documento>/<figura>.png
```

Exemplo:

```text
geometria/6ano/triangulos-elementos-e-classificacao/fig-01-elementos-do-triangulo.png
```

As fontes `.tex`, os manifestos, os conteúdos em Markdown, os blueprints e as ferramentas permanecem no repositório privado de produção. Este repositório não recebe arquivos-fonte nem conteúdo pedagógico completo.
